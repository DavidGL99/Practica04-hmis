¿Quién somos?



Somos David Gómez López y Guillermo Montalban Martinez, estudiantes de ingenieria informática en la universidad de Almería

Nos dedicamos a crear páginas web simples pero muy completas y con buen acabado visual, nos comprometemos a cumplir siempre los siguientes objetivos:

Diseñamos su página web

Somos un equipo pequeño pero estamos muy abiertos a reunirnos con el cliente para validar el desarrollo de la página y asegurarnos que se cumplen las expectativas para que el producto quede tal y como el cliente se la imaginaba

Calidad y diseño

Nos comprometemos a que nuestras páginas cumplen los estandares actuales y siguen los patrones de desarrollo responsive para que la web sea accesible desde cualquier dispositivo

Mantenimiento

Nos aseguramos de dar mantenimiento y soporte completo a las páginas web de nuestros clientes.
